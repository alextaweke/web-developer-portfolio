# Web Developer Portfolio

A personal portfolio website built with **React, TypeScript, Vite, and Tailwind CSS** to showcase my skills, projects, and experience.  

## 🚀 Features
- Responsive and modern UI with Tailwind CSS
- Animated hero section with Framer Motion
- Projects showcase with descriptions and links
- Contact form (with SMTP integration)
- Social links (GitHub, LinkedIn, Twitter, etc.)
- Dark/Light mode support (optional)

## 🛠️ Tech Stack
- **Frontend:** React, TypeScript, Vite, Tailwind CSS, Framer Motion  
- **Backend (optional):** Node.js, Express, MySQL (for contact form or admin panel)  

## 📂 Project Structure
src/
├── components/ # Reusable UI components
├── pages/ # About, Projects, Contact, etc.
├── utils/ # Helpers and motion variants
└── assets/ # Images and icons

## 📦 Installation
```bash
# Clone repository
git clone https://github.com/alextaweke/web-developer-portfolio.git

# Navigate to project folder
cd web-developer-portfolio

# Install dependencies
npm install

# Run development server
npm run dev
